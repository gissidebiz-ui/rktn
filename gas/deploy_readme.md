# デプロイ手順 (自動化スクリプト)

GASコードを複数アカウントに一括でアップロード（Push）するための機能です。

## 新規アカウントを追加する場合

1. deploy.ps1 をエディタ等で開きます。
2. $ids = @(...) の配列の中に、新しいアカウントの **スクリプトID** を追加してください。
3. 保存して閉じます。

## アップロード（デプロイ）の実行方法

VSCode内蔵のターミナルやPowerShellを開き、以下のコマンドを実行するだけです。

``powershell
.\deploy.ps1
``

これにより、リストに登録されたすべてのアカウントに対して順番に clasp push -f が実行され、一括で最新コードが反映されます。
